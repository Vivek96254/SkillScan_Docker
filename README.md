### **Run Locally**
Alternatively, you can run the frontend on your local machine:
1. Open the **React_FrontEnd** folder in **VS Code** (or any preferred editor).
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the development server:
   ```sh
   npm start
   ```
